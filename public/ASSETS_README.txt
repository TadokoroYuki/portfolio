Required Assets for Portfolio
=============================

Please add the following files to this directory:

1. favicon.ico
   - Size: 32x32 or 16x16 pixels
   - Format: ICO

2. og-image.png
   - Size: 1200x630 pixels
   - Format: PNG
   - Content: Portfolio preview image for social sharing

3. profile.jpg
   - Size: 400x400 pixels (minimum)
   - Format: JPG or PNG
   - Content: Your professional photo

4. resume.pdf
   - Format: PDF
   - Content: Your resume/CV document

After adding these files, you can delete this README.
