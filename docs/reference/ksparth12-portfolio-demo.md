## 概要

カスタムカーソルとグローエフェクトが特徴的なポートフォリオ。Shadcn UIによる高品質なコンポーネントと紫テーマ（#6366f1）を使用。

## 特徴

- **カスタムカーソル**: マウス追従カーソル、ホバー時の拡大
- **グローエフェクト**: ボタンやカードにグロー効果
- **紫テーマ**: #6366f1（Indigo）をメインカラーに使用
- Shadcn UIによる高品質なコンポーネント
