## 概要

開発者ポートフォリオのキュレーションリスト。1475以上のポートフォリオサイトが登録されており、インスピレーションを得るのに最適。PRを通じて自分のポートフォリオを追加可能。

## リストの構造

### カテゴリ分け
- **アルファベット順（A〜Z）** で整理
- 各セクションへのジャンプリンク付き
- 「Random Portfolio」機能でランダムなポートフォリオに飛べる

### エントリ形式
```markdown
- [名前](URL) [職種/専門分野]
```

例:
```markdown
- [Aakhand Tajmirul](https://www.tajmirul.site) [Frontend Engineer] - Animated
- [Brittany Chiang](https://brittanychiang.com)
```

## 本プロジェクトへの示唆

### 差別化のポイント
1. **インタラクティブ性**: 3Dアニメーション、ゲーム風UI、ターミナル風UIなど独自性のあるインターフェースが目立つ
2. **技術スタック表示**: 多くのポートフォリオが専門分野を明記しており、訪問者に専門性を即座に伝えている
3. **ユニークなコンセプト**: Whatsappスタイル、COBOLで書かれた、RAG-Powered等、技術的なストーリーがあるものが印象に残る

### 実装のヒント
1. **アニメーション**: Framer Motion等を使ったアニメーションは差別化要素
2. **テーマ**: ダークモード対応は標準的（本プロジェクトは既に対応済み）
3. **レスポンシブ**: モバイルファースト設計は必須（本プロジェクトは既に対応済み）
4. **パフォーマンス**: Vercelデプロイが多い（本プロジェクトの推奨デプロイ先と一致）

### このリストへの追加
- 本プロジェクトが完成したら、PRを通じてこのリストに追加することで、露出を増やすことが可能
- 追加時には `[Full Stack Developer]` 等の職種タグを付けると良い
